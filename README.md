# Задание 3



## Задача 1



### Текст задачи



Опишите функции, осуществляющие основные математические операции над комплексными числами (сложение, вычитание, умножение, деление, возведение в степень).



### Алгоритм решения



Программа принимает на ввод действительную и мнимую части первого числа, затем пользователь выбирает действие над комплексным числом путем ввода знака с клавиатуры.

После выбора операции, кроме возведения в степень,  происходит ввод действительной и мнимой частей второго числа.

Сложение соответственно складывает действительную и мнимую части двух чисел

Вычитание соответственно вычитает действительную и мнимую части двух чисел

Умножение: нахождение коэффициентов при мнимой и действительной частях происходит по следующей формуле: (a + bi)(c + di) = (ac - bd) + (ad + bc)i

Деление: нахождение коэффициентов при мнимой и действительной частях происходит по следующей формуле:(a + bi) ÷ (c + di) = ((ac + bd) + (bc - ad)i)÷ (c² + d²)

В случае выбора возведения в степень, программа запрашивает степень числа, затем рекурсивно умножает комплексное число на результат рекурсивного вызова от комплексного числа и степени на единицу меньше.

После ввода значений и выполнения операции программа выводит комплексное число в виде: (Действительная часть) + (Мнимая часть)i
